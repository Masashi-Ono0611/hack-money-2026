# Product Overview

**Ghost Yield** は、低稼働なEthereum L2チェーンの計算リソースをトークン化し、ガスレス裁定によってUSDC収益を生み出す財務レイヤーです。

## Core Capabilities

### 1. Compute Token (CPT) - 計算リソースの資産化
各L2の計算コストを **CPT** としてトークン化します。
- **GSU (Gas Standard Unit)**: 実行コストとデータコストを統合した標準単位 (1 CPT = 1,000,000 GSU)。
- **有効期限 (Expiration Epoch)**: ブロックスペースの時間的減衰を反映（例: `2026-Q3-Week12`）。
- **裏付け**: シーケンサーのスループットやDAスペースなど、未使用のインフラ容量が担保。

### 2. プログラム可能な価格市場 (Uniswap v4)
Uniswap v4 を用いて **CPT / USDC** 市場を構築します。
- **v4 Hook**: L2の稼働率に応じて手数料やスプレッドを動的に制御。
- **価格形成**: 「空いているL2ほどCPTが安くなる」という市場ルールを実装し、インフラの需給を価格に反映させます。

### 3. ガスレス高速裁定 (Yellow Network)
Yellow SDK のステートチャネルを活用し、L2間のCPT価格差を **ガスレスかつ高速** に裁定します。
- インテントベースの取引により、オンチェーンのコストと遅延を排除。
- 最終結果のみをオンチェーンで決済。

### 4. 収益の確定と還元 (Arc + Circle)
裁定による差益を **USDC** で決済し、L2運営者の **Vault** に集約します。
- ノード運用やシーケンサー維持などの固定費を支払える「安定資産」として還元。

## Value Proposition

**課題**: L2は「ユーザー不足 → 収益ゼロ → 固定費のみ → 撤退」という死の谷に直面しやすい。

**解決策**: ユーザーがいなくても、計算リソースの市場性（将来の予約やヘッジ需要）から収益を生み出すインフラを提供。
- **For L2 Operators**: 需要に依存しない「第2の収益源」を確保し、チェーンの生存率を高める。
- **For Traders**: ガスコストのボラティリティを取引可能な資産クラスに変え、ヘッジや投機機会を提供。
- **For Developers**: 将来のコストを固定化し、クラウドのような予実管理を可能にする。
