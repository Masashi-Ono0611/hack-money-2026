# Tech Stack

- **Core Blockchain:** Ethereum L2s (Base Sepolia, WorldCoin Sepolia)
- **Smart Contracts:**
  - Language: Solidity
  - Framework: Foundry
  - Key Integration: Uniswap v4 (Hooks), Yellow Network (Intent/Solver), Arc (Bridge/Settlement)
- **Frontend:**
  - Framework: Next.js 16 (App Router)
  - Library: React 19
  - Styling: Tailwind CSS 4
  - Formatter/Linter: Biome
  - Runtime/Package Manager: Bun (implied by bun.lockb), pnpm (root)
- **Scripting:**
  - Language: TypeScript (tsx)
